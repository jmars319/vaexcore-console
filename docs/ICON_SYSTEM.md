# VaexCore Icon System

VaexCore uses responsive app icons.

- The v3 Console brand artwork is the source of truth for the setup UI logo and icon generation.
- Large brand art includes the detailed neon scene, Console wordmark, and `CONSOLE` label.
- Small app icon slots use a center crop from the same v3 art so the `V` mark and bot/operator face stay recognizable without tiny text.

For macOS, `desktop/macOS/assets/icon.icns` uses the v3 center crop through 256 px and the full v3 Console brand artwork for 512 px and 1024 px slots.
