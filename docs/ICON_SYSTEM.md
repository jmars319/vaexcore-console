# VaexCore Icon System

VaexCore uses responsive app icons.

- Large brand art can include the detailed neon scene and wordmark.
- Small app icon slots should use the simplified `V` mark, one app glyph, no text, and minimal background detail.
- Console's small icon glyph is the bot/operator face because Console owns chat, giveaways, moderation, and live operations.

For macOS, `desktop/macOS/assets/icon.icns` uses simplified artwork through 256 px and the detailed Console brand artwork for 512 px and 1024 px slots.
